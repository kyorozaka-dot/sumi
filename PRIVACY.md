## 第1条（基本方針）

当開発者は、利用者のプライバシーを最大限尊重し、本ソフトウェア（Sumi）の利用に関連して**利用者の個人情報を一切収集しない**ことを基本方針とします。

本ポリシーは、以下の法令に準拠します。

- 個人情報の保護に関する法律（平成15年法律第57号、2022年4月1日施行の改正法を含む。以下「個人情報保護法」）
- EU一般データ保護規則 (Regulation (EU) 2016/679, GDPR)
- カリフォルニア州消費者プライバシー法 (CCPA, Cal. Civ. Code § 1798.100 et seq.)
- カリフォルニア州プライバシー権利法 (CPRA)
- 児童オンラインプライバシー保護法 (COPPA, 15 U.S.C. §§ 6501–6506)
- 個人情報の保護に関する法律施行令、規則
- 個人情報保護委員会ガイドライン

## 第2条（定義）

本ポリシーにおいて使用する用語の定義は以下のとおりとします。個人情報保護法上の定義に従います。

1. **「個人情報」**: 個人情報保護法第2条第1項に定義される情報
2. **「個人データ」**: 個人情報保護法第16条第3項に定義される情報
3. **「保有個人データ」**: 個人情報保護法第16条第4項に定義される情報
4. **「仮名加工情報」**: 個人情報保護法第2条第5項に定義される情報
5. **「匿名加工情報」**: 個人情報保護法第2条第6項に定義される情報
6. **「個人関連情報」**: 個人情報保護法第2条第7項に定義される情報
7. **「要配慮個人情報」**: 個人情報保護法第2条第3項に定義される情報
8. **「データ管理者」(Controller)**: GDPR第4条(7)に定義される者
9. **「データ処理者」(Processor)**: GDPR第4条(8)に定義される者
10. **「Anthropic社」**: Anthropic, PBC（米国デラウェア州法人）

## 第3条（取得しない情報）

当開発者は、本ソフトウェアの提供にあたり、以下の情報を**一切取得、保存、利用、第三者提供しません**。

### 3.1 個人情報及び個人データ

1. 氏名、住所、電話番号、電子メールアドレス、生年月日、性別
2. クレジットカード番号、銀行口座番号、その他の金融情報
3. 健康情報、医療情報、保険情報
4. 思想、信条、宗教、政党、職業
5. その他、個人情報保護法第2条第1項に該当する情報

### 3.2 要配慮個人情報

人種、信条、社会的身分、病歴、犯罪の経歴、被害事実その他、個人情報保護法第2条第3項に該当する情報を一切取得しません。

### 3.3 利用者の入力データ

1. 利用者が本ソフトウェアを通じて入力したテキスト
2. 利用者が変換した語句、選択した候補、変換履歴、確定履歴
3. キーストロークログ、入力タイミング情報
4. 入力先のアプリケーション情報

### 3.4 端末識別情報

1. 広告識別子（Google Advertising ID、IDFA等）
2. 端末識別子（IMEI、IMSI、MEID、シリアル番号、Android ID）
3. ハードウェア識別子（MACアドレス、Bluetooth ID）
4. SIM情報、電話番号、電話番号に紐付く情報
5. デバイス指紋（fingerprinting）に用いられる情報

### 3.5 位置情報

GPS、Wi-Fi、Bluetooth Beacon、セルラー、IPアドレス推定による位置情報を一切取得しません。

### 3.6 利用状況情報

1. アプリの起動・終了履歴、使用時間、利用頻度
2. クラッシュレポート、エラーログ、テレメトリ情報
3. クリック・タップの分析データ
4. パフォーマンスメトリクス
5. 利用者の他のアプリケーションの使用状況、インストール状況

### 3.7 通信情報

1. ブラウザ閲覧履歴、検索履歴
2. アクセス時刻、IPアドレス、リファラー
3. クッキー、Web Storage、Service Worker、その他のウェブトラッキング情報
4. ネットワーク識別情報（SSID、BSSID）

## 第4条（端末内処理）

本ソフトウェアの以下の処理は、すべて利用者の端末内で完結し、外部に送信されません。

| 処理内容 | 保存場所 | 暗号化方式 |
|---|---|---|
| かな漢字変換 | アプリ同梱辞書（読み取り専用） | — |
| アプリ設定（テーマ、フリック感度等） | Android DataStore（端末内） | OSによる |
| Anthropic APIキー | EncryptedSharedPreferences | AES256-GCM + AES256-SIV |

これらの端末内データは、個人情報保護法第16条第3項に定義する「個人データ」には該当せず、当開発者が容易に検索できる状態に体系的に構成されたものではありません。

利用者が本ソフトウェアをアンインストールした場合、上記すべてのデータは端末から完全に削除されます。

## 第5条（AI候補機能と第三者サービスの利用）

### 5.1 オプトイン方式

AI候補機能は**デフォルトで無効化**されています。利用者が以下のすべての操作を能動的に行った場合に限り、利用可能となります。

1. 設定画面でAnthropic社のAPIキーを入力する
2. 「AI候補」設定を「有効」に切り替える

### 5.2 通信内容と暗号化

上記条件を満たした場合のみ、入力中のテキストの一部（最大512文字）が、Anthropic社のAPIエンドポイント（https://api.anthropic.com）に対して、利用者自身のAPIキーを認証情報として、**TLS 1.2以上の暗号化通信**で送信されます。

### 5.3 法的位置付け（データ保護法上の役割）

AI候補機能利用時の各当事者の法的役割：

| 当事者 | GDPR上の役割 | 個人情報保護法上の役割 |
|---|---|---|
| 利用者 | データ管理者 (Controller) | 個人情報取扱事業者ではない自然人としての判断 |
| Anthropic社 | データ処理者 (Processor) | 委託先 |
| **当開発者（Sumi開発者）** | **none（伝送路提供者にとどまる）** | **none（取扱いを行わない）** |

当開発者は、利用者とAnthropic社の間の通信を媒介する技術的伝送経路の提供者にとどまり、GDPR上のデータ管理者でもデータ処理者でもありません。

### 5.4 当開発者がアクセスしないもの

当開発者は以下の情報に**一切アクセスできません**。

- 利用者のAPIキー（暗号化されて端末内のみに保存）
- AI候補機能を通じてAnthropic社に送信された内容
- Anthropic社から返信された内容
- 利用者のAnthropic社アカウントに関する情報

### 5.5 Anthropic社のプライバシーポリシー

Anthropic社による情報の取り扱いについては、同社のプライバシーポリシーをご参照ください。

https://www.anthropic.com/legal/privacy

利用者は、AI候補機能を有効化することにより、Anthropic社の利用規約及びプライバシーポリシーに同意したものとみなされます。

## 第11条（クッキー及び類似技術）

本ソフトウェアは、ネイティブAndroidアプリケーションであり、ウェブブラウザではないため、以下のウェブトラッキング技術を一切使用しません。

- HTTPクッキー (RFC 6265)
- Web Storage (localStorage, sessionStorage)
- IndexedDB
- Service Worker
- Web SQL Database
- Cache API
- HTML5 Local Storage

また、ネイティブアプリ向けの追跡技術として知られる以下も使用しません。

- 広告識別子（AAID、IDFA）の取得
- フィンガープリンティング（デバイス特性の組み合わせによる識別）
- インストール照会 (Install Attribution)
- リターゲティング技術
- クロスサイト追跡

### 11.1 改正電気通信事業法（2023年6月施行）への対応

2023年6月に施行された改正電気通信事業法第27条の12に基づく「外部送信規律」について、本ソフトウェアは以下のとおり対応します。

- 本ソフトウェアは、利用者の端末から外部の電気通信設備に対して情報送信を行いません（AI機能を有効化していない場合）。
- AI機能を有効化した場合のみ、利用者自身のAPIキーで Anthropic 社への通信が発生します。当該通信は利用者自身の意思に基づくものであり、Sumi 開発者を起点とする外部送信ではありません。
- 当開発者は、利用者の操作なくバックグラウンドで情報送信を行う仕組み（広告SDK、アナリティクスSDK、フィンガープリンティング等）を一切実装していません。

したがって、改正電気通信事業法第27条の12に基づく事前通知・公表義務の対象となる「外部送信プログラム」は本ソフトウェアには存在しません。

## 第12条（データの国際移転）

当開発者は個人情報を取得しないため、当開発者を起点とする個人データの国際移転は発生しません。

AI候補機能を有効化した場合、利用者が自らAnthropic社（米国デラウェア州法人）と通信を行います。当該通信における個人データの取扱いは、利用者とAnthropic社の間の関係であり、Anthropic社のプライバシーポリシー及び同社が締結している標準契約条項（SCC）・データ処理契約（DPA）に従います。

日本の個人情報保護法第28条に定める「外国にある第三者への個人データの提供」については、当開発者は提供者ではないため、本ポリシーにおける適用はありません。

## 第13条（Privacy by Design 宣言）

当開発者は、本ソフトウェアの設計において、Ann Cavoukian 博士が提唱し、GDPR 第25条にも採用された「Privacy by Design」の7原則を遵守します。

| # | 原則 | 本ソフトウェアでの実装 |
|---|---|---|
| 1 | **事後対応ではなく事前予防** | 個人情報を「収集後に保護」するのではなく「最初から収集しない」設計 |
| 2 | **デフォルトでのプライバシー** | AI機能はデフォルト無効、追跡機能は実装しない |
| 3 | **設計に組み込まれたプライバシー** | プライバシーは追加機能ではなくコアアーキテクチャ |
| 4 | **ゼロサムではなくポジティブサム** | プライバシーと機能性を両立（5,036語の辞書も全て端末内） |
| 5 | **エンドツーエンドのセキュリティ** | APIキー暗号化（AES256-GCM）、通信TLS 1.2以上 |
| 6 | **可視性と透明性** | 全ソースコードを GitHub で公開 |
| 7 | **利用者プライバシーの尊重** | 利用者をユーザーではなく主権者として扱う |

## 第14条（情報セキュリティ）

当開発者は、本ソフトウェアの設計において以下の情報セキュリティ措置を講じます。

1. **暗号化（保存時）**: APIキーはAES256-GCM及びAES256-SIVアルゴリズムによりEncryptedSharedPreferencesに保存
2. **暗号化（通信時）**: AI候補機能通信はTLS 1.2以上で保護、HTTPSのみを使用
3. **権限の最小化**: Android Manifest において必要最小限の権限のみを要求
4. **第三者ライブラリの審査**: 組み込む外部ライブラリは個人情報を収集しないものに限定（一覧は `THIRD_PARTY_LICENSES.md` を参照）
5. **ソースコード公開**: 全ソースコードをGitHubで公開し、第三者によるセキュリティ検証を可能とする
6. **依存関係の継続的な監視**: 既知の脆弱性を持つ依存ライブラリの使用回避

## 第15条（個人データの漏えい時の対応）

当開発者は個人データを取得しないため、当開発者を起点とする個人情報保護法第26条に定める漏えい等事案は理論上発生しません。

ただし、万一、本ソフトウェアの欠陥により利用者の端末内データが意図せず外部に送信される事象が発覚した場合、当開発者は以下の対応を実施します。

### 14.1 即時対応（事象認知後24時間以内）

1. GitHubリポジトリのSecurity Advisoriesでの公表
2. 該当する場合はアプリ内通知での緊急告知
3. 影響範囲の暫定的調査
4. 暫定的な緩和策の検討（旧バージョンの配布停止等）

### 14.2 速報対応（認知後72時間以内）

1. 個人情報保護委員会への速報（個人情報保護法第26条第1項該当の場合）
2. GDPR第33条該当の場合、所管監督機関への通知
3. 影響を受けた可能性のある利用者への通知（個人情報保護法第26条第2項）

### 14.3 確報対応（認知後30日以内、要配慮個人情報・1000人超は60日以内）

1. 個人情報保護委員会への確報
2. 修正版のリリース
3. 詳細な事案報告書の公表
4. 再発防止策の策定及び公表

## 第21条（地域別の追加開示）

### 21.1 カリフォルニア州居住者向け（CCPA/CPRA）

当開発者は、カリフォルニア州居住者の個人情報を販売（"sell"）又は共有（"share"）しません。当開発者は個人情報を取得しないため、CCPA/CPRAに基づく以下の権利についても、行使対象データが存在しません。

- 知る権利 (Right to Know)
- 削除を求める権利 (Right to Delete)
- 訂正を求める権利 (Right to Correct)
- オプトアウト権 (Right to Opt-Out of Sale/Sharing)
- 差別を受けない権利 (Right to Non-Discrimination)
- センシティブ個人情報の使用制限権 (Right to Limit Use of Sensitive Personal Information)

### 21.2 EU居住者向け（GDPR）

第20条のとおり、当開発者は個人データを取扱わないため、GDPRに基づくデータ管理者責任は発生しません。AI候補機能利用時のデータ管理者は利用者自身となります。
