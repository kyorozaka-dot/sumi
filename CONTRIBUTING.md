# Sumi へのコントリビューション

ようこそ。Sumi は無料・広告なし・プライバシー重視のキーボードを共に作るプロジェクトです。

## プルリクエストの前に

- 小さな修正は直接PRでOK
- 機能追加や大きな変更は事前にIssueで議論してください
- コードスタイルは既存に合わせてください

## 辞書への語彙追加

app/src/main/assets/dict/common.json を編集。形式：

```json
{
  "ひらがな": ["漢字候補1", "漢字候補2"]
}
```

地名・人名・専門用語などのPR歓迎。
