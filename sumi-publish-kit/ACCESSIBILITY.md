# Sumi アクセシビリティ宣言

**最終更新日:** 2026年5月25日

---

## 基本方針

Sumi は、年齢、障害の有無、利用環境にかかわらず、すべての方が快適に日本語入力を行えるよう設計することを目指しています。本宣言は、障害者差別解消法（平成25年法律第65号）、合理的配慮提供義務の趣旨を踏まえています。

---

## 準拠を目指す基準

- **JIS X 8341-3:2016**（ウェブアクセシビリティ国内標準）
- **WCAG 2.1 Level AA**（World Wide Web Consortium 国際標準）
- **Android Accessibility Guidelines**（Google公式ガイドライン）

---

## 実装済みのアクセシビリティ対応

| 項目 | 対応状況 | 詳細 |
|---|---|---|
| TalkBack（スクリーンリーダー） | ⚙️ 計画中 | キー読み上げに対応予定 |
| 文字サイズの拡大 | ⚠️ 一部 | OSの文字サイズ設定に追随 |
| 色覚バリアフリー | ✅ 対応 | 色情報のみに依存しないUI設計 |
| コントラスト比 | ✅ 対応 | WCAG AA基準（4.5:1以上）を満たす |
| キーボードナビゲーション | ✅ 対応 | 物理キーボード接続時のフォーカス対応 |
| バイブレーション補助 | ✅ 対応 | 視覚障害者向けの触覚フィードバック |
| ダークモード | ✅ 対応 | 光感受性のあるユーザー向け |

---

## 既知の課題と改善計画

1. **TalkBack 完全対応**: 全UI要素への contentDescription 付与は進行中
2. **片手モード**: 運動機能に制限のあるユーザー向け、検討中
3. **高コントラストモード**: 弱視ユーザー向け、検討中
